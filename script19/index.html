Core JavaScript Questions (Part 1)

// Section 1: Variables and Data Types

1. What are the different data types in JavaScript?
   Answer: Number, String, Boolean, Object, Null, Undefined, Symbol, BigInt

2. Explain the difference between null and undefined.
   Answer: undefined means variable declared but not assigned,
   null is an explicit assignment to represent "no value"

3. How does type coercion work in JavaScript?
   Example:
   console.log(1 + "2")      // "12"
   console.log(1 - "2")      // -1
   console.log("2" * "3")    // 6

4. What is temporal dead zone (TDZ)?
   Answer: Period between variable declaration and initialization
   where let/const variables cannot be accessed

5. Explain different ways to declare objects
   Example:
   const obj1 = {}
   const obj2 = new Object()
   const obj3 = Object.create(null)

// Section 2: Functions and Scope

6. What is function hoisting?
   Example:
   sayHello();  // Works
   function sayHello() { console.log("Hello"); }

7. How do arrow functions differ from regular functions?
   - No this binding
   - No arguments object
   - Can't be used as constructors

8. Explain lexical scope
   Example:
   function outer() {
       const x = 10;
       function inner() {
           console.log(x); // Access to outer x
       }
   }

9. What is closure and how is it useful?
   Example:
   function counter() {
       let count = 0;
       return () => ++count;
   }

10. Explain the module pattern
    Example:
    const module = (function() {
        const private = 'secret';
        return {
            public: function() {
                console.log(private);
            }
        };
    })();

// Section 3: Objects and Prototypes

11. How does prototypal inheritance work?
    Example:
    function Animal(name) {
        this.name = name;
    }
    Animal.prototype.speak = function() {
        return `${this.name} makes a sound`;
    };

12. What are getters and setters?
    Example:
    const obj = {
        _value: 0,
        get value() { return this._value; },
        set value(v) { this._value = v; }
    };

13. Explain Object.defineProperty()
    Example:
    Object.defineProperty(obj, 'id', {
        value: 1,
        writable: false,
        enumerable: true
    });

14. What is the difference between Object.freeze() and Object.seal()?
    Answer: freeze prevents modifications to properties,
    seal prevents adding/removing properties but allows modifications

15. How do you implement method chaining?
    Example:
    class Calculator {
        constructor() { this.value = 0; }
        add(n) { this.value += n; return this; }
        subtract(n) { this.value -= n; return this; }
    }

// Section 4: Asynchronous JavaScript

16. Explain the event loop
    Answer: Mechanism that handles async operations through
    callback queue and microtask queue

17. What are Promises and how do they work?
    Example:
    new Promise((resolve, reject) => {
        // async operation
        if (success) resolve(data);
        else reject(error);
    });

18. How does async/await work?
    Example:
    async function getData() {
        try {
            const response = await fetch(url);
            const data = await response.json();
            return data;
        } catch (error) {
            console.error(error);
        }
    }

19. What is the difference between setTimeout and setInterval?
    Example:
    setTimeout(() => console.log("Once"), 1000);
    setInterval(() => console.log("Repeatedly"), 1000);

20. How do you handle multiple promises?
    Example:
    Promise.all([p1, p2, p3])
    Promise.race([p1, p2, p3])
    Promise.allSettled([p1, p2, p3])
