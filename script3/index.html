JavaScript Operators Guide

Arithmetic Operators

Basic mathematical operations in JavaScript

એરિથમેટિક ઓપરેટર્સ

JavaScript માં મૂળભૂત ગણિતીય ક્રિયાઓ

let a = 10;
let b = 5;

// Addition (+)
console.log(a + b);  // 15

// Subtraction (-)
console.log(a - b);  // 5

// Multiplication (*)
console.log(a * b);  // 50

// Division (/)
console.log(a / b);  // 2

// Modulus (%)
console.log(a % b);  // 0

// Exponentiation (**)
console.log(a ** 2);  // 100

Assignment Operators

Different ways to assign values

એસાઇનમેન્ટ ઓપરેટર્સ

વેલ્યુ એસાઇન કરવાની વિવિધ રીતો

let x = 10;
x += 5;  // x = x + 5
x -= 3;  // x = x - 3
x *= 2;  // x = x * 2
x /= 4;  // x = x / 4
x %= 3;  // x = x % 3
x **= 2; // x = x ** 2

Comparison Operators

Compare values and return boolean results

કમ્પેરિઝન ઓપરેટર્સ

વેલ્યુની તુલના કરો અને બૂલિયન પરિણામ મેળવો

let a = 5, b = 10;

console.log(a == b);   // false (equal to)
console.log(a === b);  // false (strict equal to)
console.log(a != b);   // true (not equal to)
console.log(a !== b);  // true (strict not equal to)
console.log(a > b);    // false (greater than)
console.log(a < b);    // true (less than)
console.log(a >= b);   // false (greater than or equal to)
console.log(a <= b);   // true (less than or equal to)

Logical Operators

Combine conditions and perform logical operations

લોજિકલ ઓપરેટર્સ

શરતોને જોડો અને લોજિકલ ઓપરેશન્સ કરો

let isAdult = true;
let hasID = false;

console.log(isAdult && hasID);  // false (AND)
console.log(isAdult || hasID);  // true (OR)
console.log(!isAdult);          // false (NOT)

// Short-circuit evaluation
let name = "" || "Guest";  // "Guest"
let user = { name: "John" };
let userName = user && user.name;  // "John"

Unary Operators

Operators that work with a single operand

યુનરી ઓપરેટર્સ

એક ઓપરેન્ડ સાથે કામ કરતા ઓપરેટર્સ

let x = 5;
let y = -x;      // Unary negation

x++;                // Increment
console.log(x);     // 6

y--;                // Decrement
console.log(y);     // -6

// Type operators
console.log(typeof x);     // "number"
console.log(typeof "Hello"); // "string"

Conditional (Ternary) Operator

Shorthand for simple if-else conditions

કન્ડિશનલ (ટર્નરી) ઓપરેટર

સરળ if-else શરતો માટે શોર્ટહેન્ડ

let age = 20;
let status = age >= 18 ? "Adult" : "Minor";

// Nested ternary
let result = age < 13 ? "Child" 
           : age < 18 ? "Teen" 
           : "Adult";

Bitwise Operators

Perform operations on binary representations

બિટવાઇઝ ઓપરેટર્સ

બાઇનરી રજૂઆતો પર ઓપરેશન્સ કરો

let a = 5;  // 101 in binary
let b = 3;  // 011 in binary

console.log(a & b);   // 1 (AND)
console.log(a | b);   // 7 (OR)
console.log(a ^ b);   // 6 (XOR)
console.log(~a);      // -6 (NOT)
console.log(a << 1); // 10 (Left shift)
console.log(a >> 1); // 2 (Right shift)

String Operators

Concatenate and work with strings

સ્ટ્રિંગ ઓપરેટર્સ

સ્ટ્રિંગ્સને જોડો અને તેની સાથે કામ કરો

let str1 = "Hello";
let str2 = "World";

// Concatenation
console.log(str1 + " " + str2);  // "Hello World"

// Template literals
console.log(`${str1} ${str2}!`);  // "Hello World!"

// Concatenation assignment
str1 += " there";  // "Hello there"

વિવિધ ઓપરેટર્સનો ઉપયોગ કરીને સંપૂર્ણ શોપિંગ કાર્ટ કેલ્ક્યુલેટર બનાવો.

// Item prices
let prices = [29.99, 49.99, 19.99];
let quantities = [2, 1, 3];

// Calculate subtotal
let subtotal = 0;
for(let i = 0; i < prices.length; i++) {
    subtotal += prices[i] * quantities[i];
}

// Apply discount if subtotal > $100
const discount = subtotal > 100 ? 0.1 : 0;
const discountAmount = subtotal * discount;

// Calculate tax (8%)
const tax = subtotal * 0.08;

// Calculate final total
const total = subtotal - discountAmount + tax;

console.log(`Subtotal: $${subtotal.toFixed(2)}`);
console.log(`Discount: $${discountAmount.toFixed(2)}`);
console.log(`Tax: $${tax.toFixed(2)}`);
console.log(`Total: $${total.toFixed(2)}`);

Quick Quiz

Test your understanding of operators!

ઝડપી ક્વિઝ

ઓપરેટર્સની તમારી સમજ ચકાસો!

Q1: What will be the output?

let x = 5;
let y = "5";
console.log(x == y);
console.log(x === y);

પ્ર1: આઉટપુટ શું આવશે?

let x = 5;
let y = "5";
console.log(x == y);
console.log(x === y);

Output will be:

true
false

Explanation: == compares values after type conversion, while === compares both value and type.

આઉટપુટ આવશે:

true
false

સમજૂતી: == ટાઈપ કન્વર્ઝન પછી વેલ્યુની તુલના કરે છે, જ્યારે === વેલ્યુ અને ટાઈપ બંનેની તુલના કરે છે.

Q2: What's the result?

let a = true;
let b = false;
let c = true;
console.log(a && b || c);

પ્ર2: પરિણામ શું આવશે?

let a = true;
let b = false;
let c = true;
console.log(a && b || c);

Output: true

Explanation: && has higher precedence than ||
(a && b) evaluates to false, then false || c evaluates to true

આઉટપુટ: true

સમજૂતી: && ની પ્રાથમિકતા || કરતાં વધારે છે
(a && b) false થાય છે, પછી false || c true થાય છે

Q3: Predict the output

let x = 10;
let y = x++;
let z = ++x;
console.log(y, z);

પ્ર3: આઉટપુટની આગાહી કરો

let x = 10;
let y = x++;
let z = ++x;
console.log(y, z);

Output: 10 12

Explanation:
1. y = x++ assigns 10 to y, then increments x to 11
2. z = ++x increments x to 12, then assigns 12 to z

આઉટપુટ: 10 12

સમજૂતી:
1. y = x++ y ને 10 આપે છે, પછી x ને વધારીને 11 કરે છે
2. z = ++x x ને વધારીને 12 કરે છે, પછી z ને 12 આપે છે

Q4: Debug Challenge

Find and fix the errors in this code:

let price = "10";
let quantity = 2;
let total = price + quantity;
console.log("Total: $" + total);

પ્ર4: ડીબગ ચેલેન્જ

આ કોડમાં ભૂલો શોધો અને સુધારો:

let price = "10";
let quantity = 2;
let total = price + quantity;
console.log("Total: $" + total);

Fixed code:

let price = Number("10");  // or parseInt("10")
let quantity = 2;
let total = price * quantity;  // use * instead of +
console.log("Total: $" + total);

Issues:
1. String concatenation instead of multiplication
2. Price needs to be converted to number
3. Wrong operator for calculation

સુધારેલો કોડ:

let price = Number("10");  // અથવા parseInt("10")
let quantity = 2;
let total = price * quantity;  // + ની જગ્યાએ * વાપરો
console.log("Total: $" + total);

સમસ્યાઓ:
1. ગુણાકાર ની જગ્યાએ સ્ટ્રિંગ જોડાણ થયું
2. Price ને નંબરમાં કન્વર્ટ કરવાની જરૂર છે
3. ગણતરી માટે ખોટો ઓપરેટર
