// Section 2: Event Handling

26. What is event bubbling and capturing?
    Example:
    element.addEventListener('click', handler, true); // Capturing
    element.addEventListener('click', handler, false); // Bubbling

27. How do you implement event delegation?
    Example:
    parent.addEventListener('click', (e) => {
        if (e.target.matches('.child')) {
            // Handle child click
        }
    });

28. What are custom events?
    Example:
    const event = new CustomEvent('myEvent', {
        detail: { data: 'Custom data' }
    });
    element.dispatchEvent(event);

29. Explain different mouse events
    Example:
    mousedown, mouseup, click, dblclick,
    mouseover, mouseout, mousemove

30. How do you handle touch events?
    Example:
    element.addEventListener('touchstart', handleTouch);
    element.addEventListener('touchmove', handleTouch);
    element.addEventListener('touchend', handleTouch);

// Section 4: Performance Optimization

36. How do you optimize JavaScript performance?
    - Use appropriate data structures
    - Minimize DOM operations
    - Implement debouncing/throttling
    - Use web workers for heavy computation
    - Optimize loops and recursion

37. What is memoization?
    Example:
    const memoize = (fn) => {
        const cache = new Map();
        return (...args) => {
            const key = JSON.stringify(args);
            if (cache.has(key)) return cache.get(key);
            const result = fn.apply(this, args);
            cache.set(key, result);
            return result;
        };
    };

38. How do you handle memory leaks?
    - Clear intervals and timeouts
    - Remove event listeners
    - Clean up references
    - Use WeakMap/WeakSet
    - Proper closure management

39. Explain lazy loading
    Example:
    const lazyImage = new IntersectionObserver((entries) => {
        entries.forEach(entry => {
            if (entry.isIntersecting) {
                entry.target.src = entry.target.dataset.src;
            }
        });
    });
